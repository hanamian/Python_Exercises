# Index

### No. 1
**Get index**
```python
countries = ['spain', 'france', 'germany', 'norway']
capitals = ['madrid', 'paris', 'berlin', 'oslo']

# Get index of 'germany'
ind_ger = countries.index('germany')

# Use ind_ger to print out capital of Germany
print(ind_ger)
print(capitals[ind_ger])
```
Result:
```
2
berlin
```
