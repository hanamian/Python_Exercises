# Simple Arithmetic

Addition
```python
a= 5+4
print('5 + 4 = ',a)
```
Substraction
```python
b= 9-6
print('9 - 6 = ',b)
```
Multiplication
```python
c= 9*3
print('9 * 3 = ', c)
```
Division
```python
d= 9/3
print('9 / 3 = ',d)
```
### The Results must be:
```python
5 + 4 =  9
9 - 6 =  3
9 * 3 =  27
9 / 3 =  3.0  #you will always meet the float type result if you do the division like this
```
